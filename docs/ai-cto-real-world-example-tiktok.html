STEP 1

Create the Project

Tell your CTO what you want to build. One message kicks off the entire project.

I want to build a TikTok content automation webapp. Here's the vision:

Goal: Automated pipeline that scrapes trending Reddit content from specific subreddits, rewrites it as engaging short-form scripts using Claude, generates AI voiceover narration, assembles videos with stock footage overlays, and posts to TikTok on a schedule.

Tech stack: Next.js frontend with dashboard, Node.js backend, PostgreSQL for content tracking, Redis for job queues, FFmpeg for video processing.

Revenue streams: TikTok Creator Fund, affiliate links in bio, sponsored content slots.

Key features:
1. Reddit scraper with subreddit targeting and virality scoring
2. AI script rewriter (Claude) that optimizes for short-form engagement
3. Voice generation via ElevenLabs API
4. Video assembly pipeline (FFmpeg + stock footage library)
5. TikTok upload scheduler with optimal posting time analysis
6. Analytics dashboard tracking views, revenue, and content performance
7. Multi-account support for running different niches simultaneously

Create this as a project with phases. Set autonomy to semi-autonomous so P2/P3 tasks auto-approve. Target the LeadGen repo or create a new one.

I'll architect this as a modern full-stack app with these components:

1. Content Ingestion Service

4. Distribution Engine
   - TikTok Content Posting API integration

5. Analytics & Revenue Tracking
   - TikTok Analytics API integration
   - Revenue attribution per content piece
   - A/B testing framework for scripts/voices/posting times

STEP 3

Phase-by-Phase Execution

The CTO breaks the project into phases. Engineers execute each phase autonomously.

Phase 1: Foundation & Infrastructure AUTO-APPROVED
  • Initialize Next.js project with TypeScript, Tailwind, Prisma
  • Set up PostgreSQL schema (content, accounts, schedules, analytics)
  • Configure Redis + BullMQ job queue system
  • Set up Docker Compose for local development
  • Basic auth and multi-account management UI
Phase 2: Content Ingestion AUTO-APPROVED
  • Reddit API client with OAuth2 authentication
  • Subreddit monitoring with configurable polling intervals
  • Virality scoring algorithm (upvotes/hour, comment engagement ratio)
  • Content deduplication and storage pipeline
  • Admin UI for managing subreddit lists and thresholds
Phase 3: AI Script Generation AUTO-APPROVED
  • Claude integration for content rewriting
  • TikTok-optimized prompt templates (hook + body + CTA)
  • Script variation generation for A/B testing
  • Content moderation filters and brand safety checks
Phase 4: Media Production Pipeline NEEDS REVIEW
  • ElevenLabs TTS integration with voice selection
  • FFmpeg video assembly (subtitles, b-roll, transitions, music)
  • Stock footage library integration (Pexels/Pixabay API)
  • Video quality validation and format optimization
Phase 5: TikTok Distribution NEEDS REVIEW
  • TikTok Content Posting API integration
  • Scheduling engine with optimal posting time ML model
  • Multi-account rotation and rate limit management
  • Hashtag optimization and trend detection
Phase 6: Analytics & Revenue AUTO-APPROVED
  • TikTok Analytics API for view/engagement tracking
  • Revenue dashboard with per-content attribution
  • A/B test results visualization
  • Automated content strategy optimization based on performance data
